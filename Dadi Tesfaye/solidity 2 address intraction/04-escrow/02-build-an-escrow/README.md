# Practice Escrow

This section contains the six Escrow stages from the Solidity course.

Each stage is stored in its own folder with a short summary and the final `solution.sol` file.

Stages:

1. `01-addresses`
2. `02-constructor-storage`
3. `03-payable`
4. `04-approve`
5. `05-security`
6. `06-events`

The stages are cumulative, so the final `solution.sol` for each stage is the complete Escrow contract.
