# Practice Inheritance

This section contains the six staged Inheritance exercises from the Solidity course.

Each stage is stored in its own folder with a short summary and the final `solution.sol` file.

Stages:

1. `01-inherit`
2. `02-constructor-args`
3. `03-virtual-override`
4. `04-super`
5. `05-ownable`
6. `06-multiple-inheritance`

The stages are cumulative, so the final `solution.sol` for each stage includes the functionality built up to that point.
